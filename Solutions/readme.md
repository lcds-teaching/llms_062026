# Solutions

This folder contains worked versions of the lab notebooks. Each solution notebook mirrors the matching lab notebook and includes executable Python code.

Use the solutions after you have tried the lab questions yourself:

1. Open the lab and solution side by side.
2. Compare the question, `Programming Plan`, and code cell.
3. Run the solution cell.
4. Check which variable names, loops, functions, or outputs differ from your attempt.

The solutions are for learning and review, not for copying blindly. If your answer uses a different approach but produces the correct result and is easy to explain, it may still be a good answer.
