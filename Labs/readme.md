# Labs

This folder contains the lab notebooks for the course. Work through them in order:

1. `Lab_One.ipynb`: general Python foundations.
2. `Lab_Two.ipynb`: beginner NLP in Python.
3. `Lab_Three.ipynb`: remotely hosted models and APIs.
4. `Lab_Four.ipynb`: locally hosted models on modest laptops.
5. `Lab_Five.ipynb`: advanced implementation topics.

The notebooks are designed for beginners. Read the markdown guidance before each code cell, run cells from top to bottom, and use the `Programming Plan` sections to decide what to type.

Each lab is self-contained. A lab notebook should not require any solution notebook to have been opened, run, or completed first.

Most live model and API features are optional and clearly marked. Do not paste confidential health, social care, interview, or institutional data into any lab notebook.
